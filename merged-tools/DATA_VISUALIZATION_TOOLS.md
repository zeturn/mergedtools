# 数据分析与可视化工具

本次新增了 3 个强大的数据分析与可视化工具，均采用纯前端实现，无需上传数据到服务器。

## 新增工具列表

### 1. 📊 CSV/JSON 数据统计分析 (`data-stats`)

**功能特性：**
- 支持 CSV 和 JSON 两种数据格式输入
- 自动识别数值列和文本列
- 提供完整的统计指标：
  - 基础统计：总数、数值数量、均值、中位数、众数
  - 离散程度：标准差、方差、范围
  - 分位数：Q1（25%）、Q3（75%）
  - 极值：最小值、最大值、总和
- 可视化箱线图（Box Plot）展示数据分布
- 支持多列同时分析

**使用场景：**
- 快速了解数据集的基本特征
- 检查数据的分布情况和异常值
- 对比不同列的统计特性

**技术实现：**
- Clean-room 实现，无第三方依赖
- 使用原生 JavaScript 计算统计指标
- 响应式设计，支持移动端

---

### 2. 📈 图表生成器 (`chart-generator`)

**功能特性：**
- 支持 6 种图表类型：
  - 柱状图（Bar Chart）
  - 折线图（Line Chart）
  - 饼图（Pie Chart）
  - 环形图（Doughnut Chart）
  - 雷达图（Radar Chart）
  - 极坐标图（Polar Area Chart）
- 可视化配置界面，实时预览
- 支持多数据集对比
- 自定义图表标题和颜色
- 一键导出为 PNG 图片
- 内置示例数据

**使用场景：**
- 快速创建演示用图表
- 数据可视化展示
- 报告和文档中的图表制作
- 教学和学习数据可视化

**技术实现：**
- 基于 Chart.js (MIT 许可)
- 使用 react-chartjs-2 进行 React 集成
- 支持高清图片导出

---

### 3. 🔢 数据透视表增强版 (`pivot-table-advanced`)

**功能特性：**
- 支持 CSV 和 JSON 数据格式
- 灵活的行列配置：
  - 多维度行分组
  - 可选的列交叉分析
  - 动态选择值字段
- 6 种聚合函数：
  - 求和（Sum）
  - 平均值（Average）
  - 计数（Count）
  - 最小值（Min）
  - 最大值（Max）
  - 去重计数（Count Distinct）
- 自动计算行列总计和汇总
- 清晰的表格展示

**使用场景：**
- 销售数据按地区、产品、时间维度分析
- 多维度数据汇总和对比
- 快速生成数据报表
- 业务数据的交叉分析

**技术实现：**
- Clean-room 实现
- 高效的数据分组和聚合算法
- 响应式表格设计

---

## 安装依赖

这些工具需要以下依赖（已自动安装）：

```bash
npm install chart.js react-chartjs-2
```

## 访问工具

启动开发服务器后，新工具将自动出现在首页工具列表中：

```bash
npm run dev
```

访问以下路径：
- `/data-stats` - CSV/JSON 数据统计分析
- `/chart-generator` - 图表生成器
- `/pivot-table-advanced` - 数据透视表增强版

## 特色功能

1. **纯前端处理**：所有计算和处理都在浏览器中完成，数据不会上传到服务器
2. **即时反馈**：实时预览和计算结果
3. **易于使用**：直观的界面设计，内置示例数据
4. **高性能**：优化的算法，支持大量数据处理
5. **导出功能**：支持将图表导出为图片
