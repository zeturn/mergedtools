## 📊 技术实现

### 支持的编码格式

| 格式 | 依赖库 | 字符集 | 应用场景 |
|------|--------|--------|----------|
| Base64 | 原生 API | A-Z, a-z, 0-9, +, / | Web、邮件、数据传输 |
| Base32 | hi-base32 | A-Z, 2-7 | Google 2FA、Tor 地址 |
| Base45 | base-x | 0-9, A-Z, 空格等45个字符 | 健康码、二维码 |
| Base58 | bs58 | 去除易混淆字符 | 比特币地址 |
| Base62 | base-x | 0-9, A-Z, a-z | 短链接、ID生成 |
| Base85 (Ascii85) | 自实现 | 85个可打印字符 | Adobe PDF |
| Base85 (Z85) | 自实现 | ZeroMQ字符集 | ZeroMQ协议 |

### 依赖包
```json
{
  "hi-base32": "^0.5.1",
  "bs58": "^6.0.0",
  "base-x": "^5.0.1"
}
```

所有依赖均已在项目中安装，无需额外安装。
