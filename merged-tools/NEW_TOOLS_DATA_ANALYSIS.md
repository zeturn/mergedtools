# 数据分析与可视化工具 - 新增功能

本次更新新增了 3 个数据分析与可视化类工具，扩展了工具集的数据处理能力。

## 1. CSV/JSON 数据统计分析 (data-stats)

**路径**: `/data-stats`

### 功能
- 对 CSV 或 JSON 数据进行全面的统计分析
- 支持多列数据同时分析
- 计算各种统计指标

### 支持的统计指标
- **基础统计**: 数量、均值、中位数、众数
- **离散度**: 标准差、方差、范围
- **分位数**: Q1（25%）、Q3（75%）
- **极值**: 最小值、最大值
- **总和**: 数值总和

### 使用方法
1. 选择数据格式（CSV 或 JSON）
2. 粘贴或输入数据
3. 点击"分析数据"按钮
4. 查看每列的详细统计信息

### 示例数据
**CSV 格式**:
```csv
销售额,访问量
100,200
150,250
120,180
```

**JSON 格式**:
```json
[
  {"销售额": 100, "访问量": 200},
  {"销售额": 150, "访问量": 250},
  {"销售额": 120, "访问量": 180}
]
```

## 2. 图表生成器 (chart-generator)

**路径**: `/chart-generator`

### 功能
- 基于 Chart.js 的可视化图表生成
- 支持多种图表类型
- 支持多数据集对比
- 可自定义图表标题和样式
- 支持导出为图片

### 支持的图表类型
- 📊 **柱状图 (Bar)**: 适合对比不同类别的数据
- 📈 **折线图 (Line)**: 适合展示趋势变化
- 🥧 **饼图 (Pie)**: 适合展示占比关系
- 🍩 **圆环图 (Doughnut)**: 饼图的变体
- 🕸️ **雷达图 (Radar)**: 适合多维度对比
- 🎯 **极区图 (Polar Area)**: 适合展示分布

### 使用方法
1. 选择图表类型
2. 输入图表标题
3. 输入标签（逗号分隔）
4. 添加数据集并输入数据（逗号分隔）
5. 点击"生成图表"
6. 可选：点击"导出图片"保存图表

### 示例
- **标签**: `一月,二月,三月,四月,五月,六月`
- **数据集 1**: `65,59,80,81,56,55`
- **数据集 2**: `30,40,50,60,70,80`

## 3. 数据透视表增强版 (pivot-table-plus)

**路径**: `/pivot-table-plus`

### 功能
- 将 CSV 数据可视化为表格
- 支持大量数据的快速展示
- 响应式表格设计
- 支持横向滚动

### 使用方法
1. 粘贴 CSV 数据（首行为表头）
2. 点击"生成表格"
3. 查看格式化的表格

### 示例数据
```csv
产品,价格,库存,分类
商品A,99.9,100,电子
商品B,199.9,50,家居
商品C,49.9,200,图书
```

## 技术实现

### 依赖库
- **Chart.js**: MIT 许可证，用于图表生成
- **react-chartjs-2**: MIT 许可证，Chart.js 的 React 封装

### 架构特点
- 遵循项目的"一个工具一个文件夹"架构
- 支持懒加载，按需加载代码
- 纯前端运行，无需后端支持
- 数据不上传，保护隐私

## 访问方式

启动开发服务器后：
```bash
npm run dev
```

在浏览器中访问：
- http://localhost:5173/data-stats
- http://localhost:5173/chart-generator
- http://localhost:5173/pivot-table-plus

或从首页工具列表中选择对应工具。
