### 2. ⚡ Base122

**高密度 ASCII 编码**

- **字符集:** 122 个可打印 ASCII 字符
- **基数:** 122
- **分类:** 高密度编码
- **特点:**
  - 比 Base94 更高的编码密度
  - 包含更多符号字符
  - 高效率数据传输
- **用途:**
  - 高效数据传输
  - 紧凑编码存储
  - 二进制数据编码

### 3. 🔢 Base128

**7-bit ASCII 完整集**

- **字符集:** 完整的 128 个 7-bit ASCII 字符
- **基数:** 128
- **分类:** 实验性编码
- **特点:**
  - 完整覆盖 ASCII 字符集（0x00-0x7F）
  - 包含控制字符
  - 理论最高效率（7/8 = 87.5%）
- **用途:**
  - 二进制数据编码
  - 协议层编码
  - 实验性应用

## 🛠 技术实现

### Base100 实现细节

```typescript
// Emoji 字符集（100 个）
const BASE100_EMOJIS = [
  '🐀','🐁','🐂','🐃','🐄','🐅','🐆','🐇','🐈','🐉',
  // ... 100 个 Emoji
]

// 编码：每个字节 -> 一个 Emoji
const base100Encode = (bytes: Uint8Array): string => {
  return Array.from(bytes)
    .map(byte => BASE100_EMOJIS[byte % 100])
    .join('')
}

// 解码：每个 Emoji -> 对应字节
const base100Decode = (text: string): Uint8Array => {
  const emojis = [...text] // 正确处理多字节 emoji
  const bytes = emojis.map(emoji => {
    const idx = BASE100_EMOJIS.indexOf(emoji)
    return idx >= 0 ? idx : 0
  })
  return new Uint8Array(bytes)
}
```

### 字符集扩展

```typescript
// Base122 - 122 个可打印字符
base122: '0-9A-Za-z + 更多符号'

// Base128 - 完整 7-bit ASCII
base128: String.fromCharCode(0-127)
```
